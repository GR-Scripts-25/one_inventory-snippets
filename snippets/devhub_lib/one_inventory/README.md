### Inventory Setup

To use **one_inventory** with **kq_link**, follow these steps:

1. Navigate to the following directory:

   ```
   devhub_lib\modules\inventories
   ```

2. Place the **one_inventory** bridge file/folder inside the `inventory` directory.

4. go to

   ```
   devhub_lib\core\shared
   ```

5. remove the old autoDetect.lua file inside that folder 

6. Place the new autoDetect.lua file inside the folder

7. Save the file and restart the resource.


**devhub_lib** will now use **one_inventory** as its inventory system.
