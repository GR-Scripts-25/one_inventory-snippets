To use **one_inventory** with **cd_bridge**, follow these steps:

1. Navigate to the following directory: `server\inventory`

2. Add the **one_inventory** client file in the folder.

3. Then replace auto_detect.lua 

4. Save the file and restart the resource.

**cd_bridge** will now use **one_inventory** as its inventory system.
