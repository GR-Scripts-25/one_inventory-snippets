### Inventory Setup

To use **one_inventory** with **cdev_lib**, follow these steps:

1. Navigate to the following directory:

   ```
   cdev_lib\public\server
   ```

2. remove the old api.lua file inside that folder 

3. Place the new api.lua file inside the folder

4. go to

   ```
   cdev_lib\shared
   ```

5. remove the old auto_detect.lua file inside that folder 

6. Place the new auto_detect.lua file inside the folder

7. Save the file and restart the resource.

**cdev_lib** will now use **one_inventory** as its inventory system.
